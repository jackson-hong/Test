태그에 접근하는 방법

아이디로 접근, 태그명, 클래스명, name명 1. id : document.getElementById("id속성값") -> element 객체 하나 반환 2. classname : document.getElementsByClassName("class이름"); -> element 배열로 반환 3. tagname : document.getElementsByTagName("tag이름"); -> element 배열로 반환 4. name : document.getElementByName("이름"); -> element 배열로 반환
