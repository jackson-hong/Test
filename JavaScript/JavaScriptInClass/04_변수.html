변수에 대해 알아보자

변수는 데이터를 보관하기 위한 저장공간
자바스크립트에서 변수를 선언하는 방법
var 변수명;
선언시에 자료형을 선언하지 않음, 하지만 자료형은 있음
변수는 접근할 수 있는 범위에 따라 global 영역 : 기본적으로 var 변수로 선언하는 것 local 영역 : function 내부에 선언된 var 변수

자료형

자바스크립트에서 변수의 자료형을 변수에 들어오는 값(리터럴)에 의해 자료형이 결정됨. 자료형별 리터럴 표현방식을 알아보고 변수에 넣어보자.

변수의 타입 확인하기 : typeof

typeof(변수명) -> 문자형으로 자료형을 반환을 해줌
