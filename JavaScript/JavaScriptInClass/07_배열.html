배열을 선언하고 활용해보자

자바스크립트에서 배열은 자바의 컬렉션과 유사함 자바스크립트의 배열을 특정자료형에 한정하지 않고 모든 자료형을 저장할 수 있음.

배열 == 객체 -> 내장함수

indexOf() : 배열에서 요소(값)이 있는 인덱스를 리턴

concat(배열) : 두개 이상의 배열을 연결하는 기능

join() : 배열을 결합 문자열로 만들어주는 함수

reverse() : 배열의 순서를 변경

sort(정렬처리함수) : 정렬하기

배열의 데이터를 조작하는 함수

push() : 배열의 맨 뒤의 요소 추가 pop() : 배열의 맨 뒤의 요소를 제거

unshift() : 맨앞에 요소를 추가하는 함수 shift() : 맨앞에 요소를 삭제하는 함수

slice() : 배열의 일부를 잘라내서 새 배열을 만든다 splice() : 배열의 특정위치 값을 지우고, 그 위치에 값을 추가

find() : 요소를 찾는 함수 특정조건에 있는 요소 하나만 filter() : 요소를 찾는 함수 특정조건에 있는 요소들 전부

forEach() : 배열전체를 순회하는 함수
